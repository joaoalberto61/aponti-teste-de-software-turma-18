# Calculadora Simples em Python

## 🎯 Objetivo
O objetivo deste projeto é fornecer uma ferramenta simples e intuitiva para a realização de cálculos matemáticos básicos.

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem de programação principal utilizada para o desenvolvimento da lógica da calculadora.
* **PyInstaller** (ou similar): Utilizado para compilar o código fonte `.py` em um arquivo executável `.exe`.

## 📁 Estrutura das Pastas

```text
calculadora-python/

├── dist/                  # Pasta que contém o arquivo executável
│   └── calculadora.exe    # Executável do programa
│
├── src/                   # Código-fonte do projeto
│   └── main.py            # Arquivo principal
│
├── README.md              # Documentação do projeto
└── .gitignore             # Arquivos ignorados pelo Git
```

## 🚀 Como Executar o Projeto

Navegue até a pasta onde o projeto está localizado (ou até a pasta `dist`).

Dê um duplo clique no arquivo `calculadora.exe` para abrir e interagir com o programa.

# Exercício 4 – Blocos de código

Adicione exemplos de comandos utilizando blocos de código.

---

### 1. Comandos Git
Bloco de código útil para inicializar um repositório e enviar as alterações:

```bash
git init
git add .
git commit -m "feat: adicionando blocos de código ao readme"
git push origin main
```

### 2. Comandos do Terminal (Windows - CMD)
Exemplos de comandos comuns para navegar e gerenciar arquivos no prompt de comando do Windows:

```bash
:: Criar uma nova pasta
mkdir meu-projeto

:: Entrar na pasta criada
cd meu-projeto

:: Listar os arquivos e pastas do diretório
dir
```

### 3. Código em Python
Um exemplo simples de script em Python que dá as boas-vindas ao usuário:

```bash
def saudar_usuario(nome):
    print(f"Olá, {nome}! Bem-vindo ao exercício de blocos de código.")

# Executando a função
saudar_usuario("Dev")
```

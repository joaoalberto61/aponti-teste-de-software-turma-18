# Exercício 8 – Inserindo citações

Abaixo está um exemplo de como utilizar blocos de citação (`blockquote`) no Markdown para destacar frases importantes sobre tecnologia ou programação.

---

### Frase de Destaque

> "Programas devem ser escritos para pessoas lerem e apenas incidentalmente para máquinas executarem."
> 
> — *Harold Abelson*

---

*Nota: Para criar esse efeito de bloco de citação no GitHub, basta utilizar o caractere `>` antes do texto.*
